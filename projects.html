1. PLX Life Insurance: Empowering Customers with Information and Support


Created a customer-centric website for PLX Life Insurance, leveraging HTML, CSS, and JavaScript. The website is designed to empower customers by providing comprehensive information and interactive tools.


One of the key features incorporated is an insurance premium calculator. This calculator allows users to input their age, term, and desired sum assured, and it provides an instant calculation of the annual insurance premium they would need to pay. This tool assists customers in making informed decisions about their insurance coverage.


Furthermore, the website integrates a 24x7 chatbot service using Azure QnA Service and Azure Cognitive Service. The chatbot offers round-the-clock assistance, enabling customers to ask questions and receive prompt and accurate responses related to policy details, available policies, and other inquiries. This enhances the overall customer experience by providing immediate support and reducing the need for manual customer service interactions.


The website is hosted on Azure Static Web App, ensuring reliable and scalable hosting capabilities for smooth accessibility and optimal user experience.


By combining comprehensive information, an insurance premium calculator, and an interactive chatbot, the PLX Life Insurance website aims to empower customers with the necessary tools and support to make informed decisions about their insurance coverage.


Technologies Used:


  • HTML
  • CSS
  • Javascript
  • Azure Static Web App
  • Azure QnA Maker
  • Azure Cognitive Service
